# HTML

### Elementos:

- Tags são abertas com "<" e elas podem conter atributos (i.e: "class='título'")
- Atributos mudam funcionalidades ou aparências
- Por fim, temos o fechamento da tag ">"

### Estrutura básica do HTML:

- <!DOCTYPE html>
- <html>
- <head>
- <meta>
- <title></title>
- </head>
- <body>
- </body>
- </html>
